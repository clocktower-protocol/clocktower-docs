# Verificar dominio

### Advertencia

Advertencia: esta función es experimental. Úsala bajo tu propio riesgo.

### Pasos para verificar tu dominio

1. Haz clic en el botón "Cuenta" en el menú

![Verificar Dominio 0](../../../assets/edit_account_1.png)

2. Asegúrate de que hayas [agregado](./add_details) un dominio a tu cuenta.

Haz clic en el botón "Verificar Dominio"

![Verificar Dominio 1](../../../assets/verify_1.png)

3. Firma la transacción con tu billetera

![Verificar Dominio 2](../../../assets/verify_2.png)

4. Sigue las instrucciones en pantalla para cambiar tu registro DNS

![Verificar Dominio 3](../../../assets/verify_3.png)

5. Si todo está configurado correctamente en el registro DNS, el "Estado" en tu cuenta debería ser verde con "Dominio Verificado"

![Verificar Dominio 4](../../../assets/verify_4.png)
