# Ver historial

### Pasos para ver el historial

1. Selecciona la pestaña "Creadas" o "Suscrito a" dependiendo de qué suscripción quieres ver

2. Haz clic en el botón de historial

![Historial 1](../../../assets/history_1.png)

3. Ver el historial de la suscripción

![Historial 2](../../../assets/history_2.png)
