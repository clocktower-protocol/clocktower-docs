# Códigos de Error

ClockTowerSubscribe.sol

- `0` : El suscriptor no puede ser el proveedor
- `1` : Token ERC20 ya agregado
- `2` : No llamar a dirección cero
- `3` : La suscripción no existe
- `4` : La dirección del token no puede ser cero
- `5` : Token no aprobado
- `6` : Tiempo ya verificado
- `7` : Debe tener privilegios de administrador
- `8` : Debe ser el proveedor de la suscripción
- `9` : Suscriptor no suscrito o ya suscrito
- `10` : Permiso del token o saldo insuficiente
- `11` : Problema enviando reembolso
- `12` : Problema enviando tarifas
- `13` : Solo el proveedor puede cancelar la suscripción
- `14` : Debe estar entre 1 y 7
- `15` : Debe estar entre 1 y 28
- `16` : Debe estar entre 1 y 90
- `17` : Debe estar entre 1 y 365
- `18` : Cantidad por debajo del mínimo del token
- `19` : Intento de reentrancia
- `20` : Token pausado
- `21` : Por encima del límite de cancelación de suscriptores
- `22` : No hay suscriptores
- `23` : La suscripción está cancelada
- `24` : No se puede suscribir mientras se pagina
- `25` : Ya suscrito
